  <project_name>Wesołe Nutki - Preschool Website</project_name>

  <overview>
    A multilingual (Polish/English) website for &quot;Wesołe Nutki&quot; preschool located in Wrocław, Poland. The project aims to create a modern, user-friendly website that showcases the preschool&apos;s offerings, news, and activities through image galleries. Built with Hugo static site generator for easy content management by non-technical staff, the site will be deployed via GitHub Pages. The design will be based on the Educenter Bootstrap theme adapted for preschool needs, featuring a warm and welcoming aesthetic appropriate for early childhood education. Key goals include: easy content management through Markdown files and a headless CMS, fast page loads through static generation, multilingual support for Polish and English audiences, and comprehensive browser testing to ensure reliability.
  </overview>
